# Solana Bootcamp Projects

A collection of Solana programs from [Solana Developer Bootcamp 2024](https://github.com/solana-developers/developer-bootcamp-2024) I'm building while learning Solana development with Anchor framework.


## About

This repository contains various Solana smart contract projects as I progress through learning blockchain development on Solana. Each project is in its own folder with complete source code and tests.

## Projects

### 1. Favourites Program
A simple on-chain program that stores user's favorite number, movie, and games using Program Derived Addresses (PDAs).

**Features:**
- Set your favorite number, movie, and up to 4 games
- Data persists on-chain tied to your wallet
- Update your favorites anytime
- Retrieve your saved favorites
